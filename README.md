# AI-HELPER

### 说明

[**AI-HELPER**](https://github.com/cddldg/aihelper/releases/latest) 是一个各种AI功能的辅助工具。现阶段D-ID的图片说话视频功能接口的实现以及一些优化。

### 安装方法

直接到 [**Releases**](https://github.com/cddldg/aihelper/releases/latest) 中下载最新版本安装即可。
